# Second Web Design

A clean, modern starter template for web design projects.

## Features

- Responsive design with mobile-friendly layout
- Clean, modern UI with gradient backgrounds
- Smooth animations and transitions
- Ready to customize and expand

## Project Structure

```
.
├── index.html      # Main HTML file
├── style.css       # Stylesheet with responsive design
├── script.js       # JavaScript with animations
└── server.js       # Node.js static file server
```

## Getting Started

The project is ready to run! Simply edit the HTML, CSS, and JavaScript files to customize your website.

### Files You Can Edit

- **index.html** - Structure and content of your website
- **style.css** - Styling and visual design
- **script.js** - Interactive functionality

## Running Locally

The server runs on port 5000 and serves static files with cache-control headers to ensure you always see the latest changes.

```bash
node server.js
```

## Customization

Feel free to:
- Add more pages by creating new HTML files
- Include images in an `images/` folder
- Add more JavaScript functionality
- Customize colors and styling to match your brand

Enjoy building your website!
